# PyProforma v2 Examples

This folder contains examples demonstrating the v2 API for PyProforma.

## Files

- `simple_model.py` - Python script showing basic v2 model creation
- `simple_model.ipynb` - Interactive Jupyter notebook with the same example
- `temp.ipynb` - Working notebook for experimentation

## Note

The v2 API is currently in scaffolding phase. Calculation logic is not yet implemented.
